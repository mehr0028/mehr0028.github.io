About Bootstrap

Bootstrap is a free and open-source CSS platform for front-end web creation that is sensitive and mobile-first. It includes design templates for typography, shapes, buttons, navigation, and other interface elements that are based on CSS and (optionally) JavaScript.

Bootstrap Icon

For the first time, Bootstrap has its own open source SVG icon library, which is specifically designed to work with our components and documentation. Bootstrap Icons are best used with Bootstrap elements, but they can be used in any project. They're SVGs, which means they'll scale quickly and easily, can be used in a variety of ways, and can be styled with CSS.

Official Theme

Premium Bootstrap themes from the official Bootstrap Themes marketplace will take your project to the next level. Themes are built on top of Bootstrap as standalone frameworks, complete with new components and plugins, documentation, and powerful build tools.

Different Headings

Bootstrap describes the basic display, typography, and connection styles for the entire web.Traditional heading elements are best used in the main content of your website. Consider using a show heading—a wider, slightly more opinionated heading style—when you need a heading to stand out.

Great Columns

With our flexbox grid method, you can change columns with a variety of options for spacing, ordering, and offsetting. Also, learn how to handle the widths of non-grid components with column groups. Columns are based on the flexbox architecture of the grid.All material is put in columns when designing grid templates. Predefined classes in Bootstrap make it easy to build quick, sensitive templates.

Variety Buttons
